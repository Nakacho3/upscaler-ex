# AIマスク生成機能 変更内容確認書 (Walkthrough)

本機能は、画像内の要素（人物、顔、オブジェクト、テキスト、空、背景など）をAI（MediaPipe および OpenCV）を用いて自動的に解析し、セグメンテーションマスクを生成する機能です。

## 実施した変更内容

### 1. マスク生成推論エンジンの新規実装
- **[ai_mask_engine.py](file:///h:/My%20Apps/upscaler%20ex/ai_mask_engine.py) [NEW]**
  - **Person (人物)** / **Background (背景)**: MediaPipe Selfie Segmentation による抽出。
  - **Face (顔)**: MediaPipe Face Detection による顔領域の検出およびマスク化。
  - **Text (テキスト)**: OpenCV MSER（最大安定極値領域）を活用した文字領域検出。
  - **Sky (空)**: HSV色空間（青空/白い空のしきい値処理）と位置情報を組み合わせた簡易抽出。
  - **Object (オブジェクト)**: 前景部のエッジ検出（Canny）および輪郭領域から抽出したその他オブジェクト。
  - 各レイヤーマスクの半透明色分けオーバーレイ合成機能 (`create_overlay`)。

### 2. UIのタブ化とAIマスク画面の構築
- **[main.py](file:///h:/My%20Apps/upscaler%20ex/main.py) [MODIFY]**
  - ウィンドウ全体を `ctk.CTkTabview` によるタブ構成に変更（「アップスケール」と「AIマスク生成」タブ）。
  - **MaskPreviewCanvas [NEW]**: マスクプレビュー用のカスタムキャンバス。ドラッグによるパン移動、マウスホイールによる拡大縮小に対応。
  - AIマスク側サイドパネルの構築（使用GPU情報、検出感度スライダー、各カテゴリレイヤーのトグルのチェックボックス、信頼度スコア表示）。
  - バックグラウンド解析スレッドによる非同期処理、プログレスバーによる進行状況通知。
  - 画像ロード時および画像クリア時の連動初期化。
  - **マスク保存**: 指定フォルダに有効なカテゴリのマスク画像を「`[画像名]_[カテゴリ名]_mask.png`」として一括エクスポートする処理。

## ユーザー動作確認手順（お願い）
ローカルPC環境でターミナルを実行できないセキュリティ制限がかかっているため、お手数ですが以下の手順で動作をご確認ください。

1. プロジェクトフォルダにある `run_app.bat` を実行して、アプリケーションを起動します。
2. アプリの最上部に「アップスケール」「AIマスク生成」というタブが表示されることを確認します。
3. 任意の画像をドラッグ＆ドロップ、または「別の画像を開く」でロードします。
4. 「AIマスク生成」タブに切り替え、「⚡ AIマスク解析を開始」ボタンを押下します。
5. 解析完了後、プレビュー画像上に半透明の色分け（例: 人物は赤、顔は緑）でマスクがオーバーレイ表示され、各レイヤー名の横に信頼度スコア（例: 信頼度: 0.95）が表示されることを確認します。
6. レイヤーチェックボックスを切り替えた際、オーバーレイ表示がリアルタイムで更新されることを確認します。
7. 「💾 解析結果(マスク)を保存」ボタンから、任意のフォルダにマスク画像を書き出せることを確認します。
